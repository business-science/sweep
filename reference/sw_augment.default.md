# Default augment method

By default,
[`sw_augment()`](https://business-science.github.io/sweep/reference/sw_augment.md)
uses
[`broom::augment()`](https://broom.tidymodels.org/reference/reexports.html)
to convert its output.

## Usage

``` r
# Default S3 method
sw_augment(x, ...)
```

## Arguments

- x:

  an object to be tidied

- ...:

  extra arguments passed to
  [`broom::augment()`](https://broom.tidymodels.org/reference/reexports.html)

## Value

A tibble generated by
[`broom::augment()`](https://broom.tidymodels.org/reference/reexports.html)
