# Print the ARIMA model parameters

Refer to forecast:::arima.string. [`forecast`
arima.R](https://github.com/robjhyndman/forecast/blob/master/R/arima.R)

## Usage

``` r
arima_string(object, padding = FALSE)
```

## Arguments

- object:

  An object of class Arima

- padding:

  Add padding to the name returned
