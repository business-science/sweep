# Default glance method

By default,
[`sw_glance()`](https://business-science.github.io/sweep/reference/sw_glance.md)
uses
[`broom::glance()`](https://broom.tidymodels.org/reference/reexports.html)
to convert its output.

## Usage

``` r
# Default S3 method
sw_glance(x, ...)
```

## Arguments

- x:

  an object to be tidied

- ...:

  extra arguments passed to
  [`broom::glance()`](https://generics.r-lib.org/reference/glance.html)

## Value

A tibble generated by
[`broom::glance()`](https://broom.tidymodels.org/reference/reexports.html)
